# Intelligent Banking Assistant  

## Overview  
This project involves fine-tuning the Llama 3 model on a 50,000-record banking dataset to develop an intelligent banking assistant. The system achieves >90% accuracy in understanding and responding to customer queries.  

## Technologies Used  
- Python  
- PyTorch  
- Hugging Face Transformers  
- Google Cloud Platform (GCP)  

## Key Features  
- Fine-tuned Llama 3 for Persian language and banking-specific tasks.  
- Optimized for low-latency inference in production environments.  
- Integrated with a chatbot interface for seamless customer interactions.
